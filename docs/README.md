This is a passion project made for my text-based RPG I play with my friends.

Main Features:

- Character status (HP, Mana, XP, Inventory, Skills, Abilities, Spells, Tamed Monsters, Level-up Handler)
- Real-Time Map (Markers, Current Locations, Travel Time Calculator, Fog of War)
- Community Wiki (Post Categories, Private/Public Posts)
